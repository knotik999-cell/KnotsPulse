# Knots Pulse — кликер от Кнота

Неоновый кликер с табами, мини-играми и достижениями в стиле Steam-тостов. Проект написан на чистом HTML/CSS/JS и работает прямо из браузера — нативный SPA без сборщиков.


## Механики

- **Pulse Core** — отдельная страница, где живёт кнопка-кнопульс, счётчик сессии и ручной прогресс.
- **Лавка импульсов** — энергетические, кредитные и дата-апгрейды с прогрессирующей ценой и таймерами.
- **Лавка искр** — магазин за очки перерождений: клики ×1.2, пассив ×1.25, глобальный множитель и прирост данных.
- **Игрульки лаборатории** (отдельный экран):  
  - *Flux Dice* — риск-кредиты.  
  - *Echo Trail* — челлендж на 10 секунд кликов.  
  - *Oracle Pulse* — мистические баффы за энергию.  
  - *Glitch Lock* — memory puzzle с кодом.  
  - *Nova Drift* — мини-аркада с бегунком и точностью.  
- **Перерождения** — конвертация каждых 5 000 кредитов в искру, сброс цикла и сохранение достижений/модов.
- **Достижения** — всплывающие тосты справа-снизу, новые трофеи за мини-игры и перерождения.
- **Сохранения** — авто+ручные, все данные в `localStorage`.

## Пасхалки

- Кнопка `???` и заметка `6-6-8-6-8-8-4` подсказаны для комбо.
- Konami Code даёт пачку данных и ачивку.
- Некоторые карточки лавки меняют поведение интерфейса (например `Подпись Кнота`).

## Структура

```
├── index.html   # разметка и структура интерфейса
├── style.css    # визуал: градиенты, стеклянные панели, тосты достижений
└── script.js    # логика игры, мини-игр, ачивок и сохранений
```
